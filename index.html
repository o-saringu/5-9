プログラム復習コンテンツ

mouseButton変数

サンプルコードの結果

サンプルコード

　　 void setup(){

　　 　size(120, 120);

　　 　strokeWeight(30);　　 　

　　 }

　　

　　 void draw(){

　　 　background(204);

　　 　stroke(102);

　　 　line(40, 0, 70, height);

　　 　if(mousePressed){

　　 　　if(mouseButton==LEFT){

　　 　　　stroke(255);

　　 　　}else{

　　 　　　stroke(0);

　　 　　}

　　 　　line(0, 70, width, 50);

　　 　}

　　 }

説明文

　マウスに2個以上のボタンがあればどのボタンを押したかを判別することができる。

　変数の値はLEFT、CENTER、RIGHTの３つであり、それぞれ左、中央、右となる。

　押されたボタンを判別するには==演算子を使用する。
